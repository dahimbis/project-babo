# Project Babo
 
A small app that turns any Bible passage into a short animated story.
 
You type in a verse reference. The app pulls the passage, breaks it into 4 to 6 short scenes, draws each one as an original minimalist black-and-white line drawing, plays it back as a slideshow with camera moves and gentle fades, narrates each scene out loud, and adds background ambience that matches the mood. You can record the whole thing and download it as a video.
 
The drawings are all stick figures with thick black ink lines, plain white backgrounds, no color. Bible storybook style.
